# 🤖 ESP32 Desktop Companion with Home Assistant Intergration

A custom-built interactive desktop gadget powered by ESP32, designed as a fun and expressive companion device with smart home integration.

This project combines hardware, firmware, and 3D design into a single embedded system.

---

## 🚀 Tech Stack
- ESP32
- C++ (PlatformIO / VSCode)
- MQTT (Home Assistant integration)

---

## 📦 Hardware Components
- ESP32 microcontroller  
- 1.2" OLED display (SH1106)  
- Capacitive touch sensor  
- Temperature & humidity sensor  
- Buzzer  
- Custom 3D printed enclosure  

---

## 🧩 Features
- Animated “eyes” using RoboEyes library  
- Touch-based interaction system  
- Sound feedback (custom RTTTL tones)  
- Smart home control via Home Assistant  
- Real-time MQTT communication  
- Environmental sensing (temperature & humidity)  

---

## 🛠️ Libraries Used
- Adafruit GFX Library  
- Adafruit SH110X  
- SH1106
- AnyRtttl (custom sound generation)  
- PubSubClient (MQTT)  
- RoboEyes (for animated expressions)  

---

## 🎮 Functionality
- Interactive touch input triggers actions & animations  
- Custom sound effects on button presses and notifications  
- Sends MQTT commands to control smart home devices  
- Displays expressive animations on OLED screen  
- Designed as a responsive desktop companion  

---

## 🏠 Smart Home Integration
Integrated with Home Assistant using MQTT, allowing the device to:
- Trigger automations via touch input  
- Act as a physical control interface for smart devices  
- Provide feedback through sound and visual cues  

---

## 🧠 Concept
The goal of this project is to create a playful, interactive gadget that blends personality with functionality — acting as both a decorative desk companion and a smart home controller.

---

## 🖨️ Hardware Design
- Custom-designed and 3D printed enclosure  
- Compact form factor for desktop use  
- Designed to house all components cleanly
